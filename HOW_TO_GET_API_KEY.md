# How to Get Google Cloud Vision API Key

## Step 1: Enable the API (do this first!)

1. Go to: https://console.cloud.google.com/
2. In the top search bar, type: **"Cloud Vision API"**
3. Click on **"Cloud Vision API"** from the results
4. Click the **"Enable"** button (if not already enabled)

## Step 2: Create API Key

1. In Google Cloud Console, look at the left sidebar
2. Click on **"APIs & Services"**
3. Click on **"Credentials"** (it's under APIs & Services)

## Step 3: Create API Key

1. On the Credentials page, look for **"Create Credentials"** button (top of page)
2. Click it
3. Select **"API key"** from the dropdown

## Step 4: Copy the Key

1. A popup will appear with your new API key
2. **Copy it** (click the copy button or select all and copy)
3. Send it to me!

## Alternative: Restrict the key (optional but recommended)

After creating the key, you can restrict it to only Vision API:
1. Click on the API key in the list
2. Under "API restrictions", select "Cloud Vision API"
3. Click Save
